# 🌐 Cognifyz Web Development Internship - Level 1 Task 1

## 📌 Project Overview

This project is part of the **Cognifyz Web Development Internship Program**.
I created a simple and clean webpage using **HTML and CSS**, including a navigation menu, content section, image display, and a basic form.

---

## 🎯 Task Objective

The objective of this task is to understand the fundamentals of web development by building:

* A webpage with **heading, paragraph, and image**
* A **navigation menu** with three links
* A **basic form** with input field and submit button

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## ✨ Features

✔ Navigation bar with 3 links (Home, About, Contact)
✔ Clean and simple UI design
✔ Image displayed using local file
✔ Input form with submit button
✔ Basic styling using CSS
